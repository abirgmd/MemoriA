## 🚀 How to Start the System

### Option 1: Automated (Recommended)
```powershell
# Windows
cd MemoriA-planning
.\START_ALL_SERVICES.ps1
```

```bash
# Linux/Mac
cd MemoriA-planning
chmod +x START_ALL_SERVICES.sh
./START_ALL_SERVICES.sh
```

### Option 2: Manual
Start 5 terminals and run in each:

**Terminal 1:**
```bash
cd MemoriA-dev/MemoriA-Registry
mvn spring-boot:run
```

**Terminal 2:**
```bash
cd MemoriA-dev/MemoriA-Planning-Service
mvn spring-boot:run
```


## 📞 Troubleshooting

### Services won't start?
1. Ensure MySQL is running on port 3307
2. Check if ports 8091, 8092, 8888, 8761, 4200 are available
3. Verify Java 17+ and Node 18+ installed
4. Check application.yml files have correct database credentials

### Frontend can't reach API?
1. Check environment.ts has `apiUrl: 'http://localhost:8888'`
2. Verify Gateway is running on port 8888
3. Open DevTools and check Network tab
4. Verify Eureka shows services registered

### Services not registering with Eureka?
1. Check each service logs for Eureka connection errors
2. Verify Eureka is running on port 8761
3. Check application.yml Eureka configuration
4. Look for "Successfully registered with Eureka" in logs

See MICROSERVICES_INTEGRATION_COMPLETE.md for more troubleshooting tips.
